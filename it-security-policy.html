IT & Security Policy

This page outlines key information security and IT hygiene practices adopted in relation to the Innova Hi-Tech website.

Secure Website Access

The Innova Hi-Tech website is intended to be served over HTTPS to help protect data in transit between visitors and the site. Visitors are encouraged to access the site using modern browsers that support current security standards.

Minimal Data Collection

The public website is designed to collect only limited contact information required to respond to enquiries. No payment card data is collected and no online transactions for firearms or air guns are conducted via this site.

System and Application Hygiene

Underlying systems and applications are maintained with regular updates and security patches wherever applicable. Access to administrative interfaces is restricted to authorised personnel using controlled credentials and, where feasible, additional authentication safeguards.

Logging and Monitoring

Basic logging may be used to support troubleshooting, availability monitoring, and security review. Logs are handled in accordance with internal retention practices and are not used to build marketing profiles.

Vulnerability Management

Innova Hi-Tech aims to identify and remediate material vulnerabilities affecting the public website in a timely manner. Where appropriate, external experts or auditors may be engaged to assist in strengthening the security posture.

Responsible Use

Users are requested not to attempt to circumvent security controls or perform any testing that could impact the availability, integrity, or confidentiality of systems or data. If you believe you have identified a security concern related to this website, you may report it using the published contact details.
